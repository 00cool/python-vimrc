# .VIMRC
VIM Configuration


### Installation guide

- Put `.vimrc` file to home dir.

```bash
cd
git clone https://github.com/rmk135/vimrc.git
cp vimrc/.vimrc ~/.vimrc
```

- Setup Vundle.

```bash
git clone https://github.com/gmarik/Vundle.vim.git ~/.vim/bundle/Vundle.vim
```

- Run VIM and install plugins

```bash
vim

# inside VIM
:PluginInstall
```

- Use VIM!
