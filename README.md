# vimrc
VIM Configuration


Installation guide
==================

1. Put `.vimrc` file to home dir.

```bash
cd
git clone https://github.com/rmk135/vimrc.git
cp vimrc/.vimrc ~/.vimrc
```

2. Setup Vundle.

```bash
git clone https://github.com/gmarik/Vundle.vim.git ~/.vim/bundle/Vundle.vim
```

3. Run VIM and install plugins

```bash
vim

# inside VIM
:PluginInstall
```

4. Use VIM!
